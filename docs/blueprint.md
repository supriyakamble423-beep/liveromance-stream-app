# **App Name**: GlobalStream

## Core Features:

- Live Stream Display: Showcase live video streams from global hosts.
- Host Categorization: Categorize hosts based on interests and trends using Firestore data.
- Real-time Chat: Enable real-time chat between viewers and hosts during live streams.
- Face Verification: Verify host identities using AI-powered face recognition to ensure platform safety.
- Location Based Matching: Display host profiles on a world map, for an intuitive user experience
- AI Auto-reports: The system analyzes error reports to identify potential safety issues based on context tool
- Host Referral Program: The system keeps track of which coins each host earns, so referrers are properly awarded 1%.

## Style Guidelines:

- Primary color: Deep blue (#3B82F6) for trust and stability.
- Background color: Light gray (#F9FAFB) for a clean, modern feel.
- Accent color: Vibrant orange (#FF7043) for highlighting key elements and calls to action.
- Body and headline font: 'Inter', a sans-serif font, for clear, readable text.
- Use Material Symbols for a consistent, modern look.
- Card-based layout to display host profiles, categories, and live streams.
- Subtle transition animations when navigating between pages and displaying live streams.
